History

Created by Douglas Petla as a personal project, Menote is a fast and productive journaling and note-taking application. This Single Page Application has its front-end running with Angular 9 and material design to create a fast, clean, and responsive UI, with a real-time database powered by Firebase. Menote is a Progressive Web App (PWA) that provides great user experience, loads instantly and works offline.

Douglas Petla is a developer living and working in Toronto. He spends his time developing front end (HTML, CSS, and JavaScript/Typescript) applications, while trying to cross items off his bucket list of things he wants to experiment with such as React, GraphQL, Go, Node, and others.
